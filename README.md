# Where Champions Come From

An interactive 3D globe mapping every medalist from the Milano-Cortina 2026 Winter Olympics to the city they grew up in.

**Live:** [hyperloco-llc.github.io/olympics-2026](https://hyperloco-llc.github.io/olympics-2026/)

Built with Mapbox GL JS. ~350 athletes across 30+ countries, connected by arc lines to Cortina d'Ampezzo.
